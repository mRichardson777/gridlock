## Gridlock — Project Summary

### Concept
Mobile Rush Hour clone for iPhone. Daily puzzle challenge with social sharing.

### Core Mechanic
- 6x6 grid (like the physical Rush Hour board game)
- Main piece (red car) must slide out through exit on the right
- Blocking pieces (cars = 2 cells, trucks = 3 cells) slide along their axis only
- Horizontal pieces move left/right, vertical pieces move up/down

### Game Mode
- Daily 90-second sprint — solve as many of that day's puzzles as possible
- Same puzzles for ALL users each day (like Wordle)
- Puzzles increase in difficulty throughout the day's set
- Shareable results (score card, like Wordle)
- Streak system — play daily to maintain streak

### Monetization Plan
- **Free core**: daily 90-sec challenge, sharing, streaks — never paywalled
- **Rewarded ads**: watch ad → get a hint or +15 seconds
- **Gridlock Pro**: $2.99/mo or $19.99/yr — no ads, practice mode, full stats, early access
- **Puzzle Packs**: $0.99–$2.99 one-time themed packs
